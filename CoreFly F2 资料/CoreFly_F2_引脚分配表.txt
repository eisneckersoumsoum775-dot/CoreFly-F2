八、修改对照表 (原理图中需要改动的网络)
═══════════════════════════════════════════════════════════════════════════

 位置         原接法              改为
───────       ──────              ──────
 气压计 SCL   PB10              → PB6 (I2C1_SCL)
 气压计 SDA   PB11              → PB7 (I2C1_SDA)

 LoRa SCK     PB3 (原共用)       → PC10 (SPI3_SCK)
 LoRa MISO    PB4               → PC11 (SPI3_MISO)
 LoRa MOSI    PB5               → PC12 (SPI3_MOSI)
 LoRa NSS     PC4               → PD6
 LoRa DIO0    PC5               → PD7
 LoRa NRESET  — (未接)          → PD4

 Flash MISO   PB14              → PC2 (腾出 TIM8_CH2N)
 Flash MOSI   PB15              → PC3 (腾出 TIM8_CH3N)

 接收机 RX    — (未接 MCU)      → PD9 (USART3_RX)
 接收机 TX    — (未接 MCU)      → PD8 (USART3_TX)

 蜂鸣器 BEEP  — (未接 MCU)      → PE0
 LED 红       — (未接 MCU)      → PD2
 LED 蓝       — (未接 MCU)      → PD3

 VBAT_SENSE   — (未接 MCU)      → PC0 (ADC)

 ESC1 LIN3    PE6 (旧分配)      → PE8 (TIM1_CH1N)
 ESC3 LIN1    PE5               → PB11 (TIM2_CH4)
 ESC3 LIN2    PE6               → PE5 (TIM9_CH1)
 ESC3 LIN3    PD12              → PE6 (TIM9_CH2)
